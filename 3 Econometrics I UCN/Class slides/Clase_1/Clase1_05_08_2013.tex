\section{Qué es la econometría}
\begin{frame}
\frametitle{Qué es la econometría}
\small
\begin{itemize}
\item <1> \textbf{G. Tintner:} \textcolor[rgb]{1.00,0.00,0.00}{la econometría consiste en la aplicación de la teoría económica, la matemática y los métodos estadísticos a los datos económicos para establecer resultados numéricos y verificar las leyes económicas}.\\
\bigskip
\item <1> \textbf{W.C. Hood y T.C. Koopmans:} la econometría es una rama de la economía donde la teoría económica y los métodos estadísticos se fusionan en el análisis de los datos numéricos e institucionales.\\
\bigskip
\item <1> \textbf{T. Havellmo:} el método de la investigación econométrica intenta, esencialmente, unir la teoría económica y las mediciones reales, empleando la teoría y la técnica de la inferencia estadística como un puente.\\
\bigskip
\item <1> \textbf{O. Lange:} la econometría es la ciencia que trata de la determinación, por métodos estadísticos, de leyes cuantitativas concretas que rigen la vida económica. Esta combina la Teoría Económica con la Estadística Económica y trata de dar, por métodos matemáticos y de inferencia, una expresión concreta a las leyes de la teoría.
\end{itemize}
\end{frame}

\section{Metodología de la econometría}
\begin{frame}
\frametitle{Metodología de la econometría}
\small
\begin{itemize}
\item <1> Formular una hipótesis o teoría.
\item <1> Obtener los datos.
\item <1> Especificación del modelo: consiste en usar la teoría, leyes o hipótesis particulares económicas, para investigar las relaciones entre variables y agentes de la economía.
\item <1> Estimación del modelo: trata de la utilización de instrumentos auxiliares como las matemáticas y la estadística para estimar el modelo objetivo.
\item <1> Verificación del modelo: en esta etapa se efectúa la interpretación económica
del modelo estimado y se realizan pruebas estadísticas. La fase de verificación tiene un papel muy importante dado que examina si la expresión cuantificada puede utilizarse adecuadamente con base en la teoría económica.
\item <1> Predicción: el modelo obtenido puede ser utilizado para la predicción y el desarrollo de muchas aplicaciones. Pueden surgir nuevos resultados teóricos, y generarse implicaciones de política económica a partir de las conclusiones del modelo.
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Ejemplo}
\small
\begin{itemize}
\item <1> (Teoría de consumo keynesiana) Keynes observó que, como regla general y en promedio, las personas consumen más cuando aumentan
sus ingresos, pero que ese aumento en el consumo es menor que el aumento en los ingresos.
\item <1> (Datos) Datos de ingreso disponible y de gastos en consumo. Tipos de datos: series temporales, datos de sección cruzada, datos
agrupados.
\item <1> (Modelo econométrico) El modelo matemático es exacto o determinístico, pero en la realidad las relaciones económicas no son
exactas. El económetra agrega entonces un término de error aleatorio ($\varepsilon$), qué hay detrás de este error? La intuición es que este error contiene todos los factores que no podemos observar.
\begin{equation}
Y=\beta_{0}+\beta_{1}X+\varepsilon
\end{equation}
Estableciendo: $0< \beta_{1} <1$
\end{itemize}
\end{frame}


\begin{frame}
\frametitle{Ejemplo}
\small
\begin{itemize}
\item <1> (Estimación) El objetivo es estimar los parámetros del modelo. En este caso, usaríamos técnicas de análisis de regresión simple, como por ejemplo, método de Mínimos Cuadrados Ordinarios. Las estimaciones de los parámetros son  beta 0 y beta 1 y el modelo estimado será:
\begin{equation}
\hat{Y}=\hat{\beta_{0}}+\hat{\beta_{1}}X+\varepsilon
\end{equation}
\item <1> (Contrastar que el modelo esté bien especificado) IDEA: responder a la pregunta: ¿qué validez tiene nuestro modelo? (porque, por
ejemplo, puede haber otros factores importantes que influyan en las decisiones de consumo).
\item <1> (Predicción) Si conformamos las hipótesis, podemos usar los valores estimados de los coeficientes (o parámetros) para predecir el valor del consumo para un aumento de ingreso dado. Eventualmente, supongamos que un gobierno quiera mantener el nivel de consumo agregado sobre cierto valor mínimo. Luego, conociendo que la elasticidad es de aproximadamente 0.72, podría controlar el nivel de ingreso (a través de subsidios o impuestos) para obtener un nivel deseado de consumo.
\end{itemize}
\end{frame}


\section{Qué es un modelo?}
\begin{frame}
\frametitle{Qué es un Modelo?}
\begin{block}{\textbf{Modelos científicos}}
Un modelo es una representación simplificada de la realidad. Los investigadores y los profesionales de diversas áreas del conocimiento trabajan con éstos esquemas, los cuales les permiten estudiar el comportamiento de un fenómeno de interés.
\end{block}

\begin{block}{\textbf{Modelo económico}}
Se denomina modelo económico a cualquier conjunto de supuestos que describen
una economía o parte de una economía.
\end{block}

\begin{block}{\textbf{Modelo econométrico}}
El modelo econométrico es el modelo económico que contiene las especificaciones necesarias para su validación empírica. Es usual concebir el modelo econométrico como un modelo conformado por una parte determinística y una parte aleatoria o término de error.
\end{block}
\end{frame}


\begin{frame}
\frametitle{Características mínimas de un modelo}
\begin{itemize}
\item <1> Que represente un fenómeno económico real.
\item <1> Que la representación sea simplificada, y.
\item <1> Que se haga en forma matemática.
\end{itemize}

\textbf{¿Por qué los modelos econométricos contienen perturbaciones aleatorias?}
\bigskip
\begin{itemize}
\item <1> Datos: en muchos casos el grado de control que se puede tener sobre las variables de interés es bajo.
\item <1> Número de variables: principio de parsimonia
\item <1> Disponibilidad de información: muchas veces cuando el investigador quiere incluir una variable importante en el modelo se encuentra con la limitación de cómo cuantificarla.
\item <1> Forma funcional: un investigador puede postular que la relación entre las variables de un modelo es de tipo lineal; no obstante, otro investigador podría formular una especificación funcional distinta, por ejemplo cuadrática.
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Elementos que componen un modelo}
\small
\begin{itemize}
\item <1> Variables: característica de una población que puede tomar diferentes valores.
\item <1> Ecuaciones: es una igualdad conformada por una expresión matemática que establece relaciones entre variables.
\item <1> Parámetros: actúan como factores de ponderación de cada variable explicativa y miden el efecto de las fluctuaciones de estas variables sobre la variable dependiente.
\end{itemize}
\end{frame}
